-Unzip the histopathologic-cancer-detection.zip archive: it creates a folder of the same name
-Transfer the LympNodeCancerClassifierv2.ipynb jupyter notebook in it
-Open your anaconda environment, navigate to the Week3Repo folder and run the notebook
